Checklist Item
Multi-Device CSV Template

The CSV must have exactly three columns. Each row sets the flow rate for a specific device at a specific time — the controller interpolates linearly between rows. Timestamps must be ISO-8601 UTC (e.g. 2026-03-11T08:00:00Z).

Emission ID
Device identifier — must match the device name configured in the controller (e.g. MR-01).
Time (UTC)
ISO-8601 timestamp in UTC. Rows for the same device must be in chronological order.
Flow (SLPM)
Target flow rate in standard litres per minute. Must be a non-negative number within the device's range.
